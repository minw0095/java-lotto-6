### 입력

* 로또 금액을 입력받는다.(1000원 단위)

  - 1000원 단위인지 확인

  - 숫자인지 확인

    

* 당첨 번호를 입력받는다.

  - 6개의 숫자가 모두 다른지 확인

  - 쉼표(,)를 기준으로 구분

    

* 보너스 번호를 입력 받는다.



### 출력

* 입력받은 금액에 맞는 갯수를 구매한다.
  - 구매 갯수를 출력한다
  - 서로 다른 6개의 숫자를 구매 갯수 만큼 출력한다.



* 입력받은 당첨번호와 구매한 번호와 일치 하는지 출력한다.
  - 당첨은 맞은 갯수가 3~6개 사이일 때 이다.
  - 통계 마지막에는 수익률을 넣는다.
